# 中国十大热门旅行景点攻略网站 — 需求文档

## 1. 项目概述

构建一个面向国内外游客的中国十大热门旅行景点攻略网站，提供每个景点的详尽攻略信息，
包含交通、门票、路线、美食、住宿等实用内容，以清晰、美观的页面形式呈现。

目标受众: 自助游爱好者、家庭出游用户、外国游客（支持中英文）

## 3. 每个景点攻略页面包含的信息模块

每个景点拥有独立详情页，包含以下信息模块:

### 3.1 景点概览 (Hero 区域)
- 景点名称（中英文）
- 封面大图 / 轮播图（3-5 张）
- 一句话标志性描述
- 综合评分（星级 1-5）
- 热度标签（如 "世界遗产"、"5A 景区"、"网红打卡"）

### 3.2 基本信息
- 详细地址
- 开放时间（含旺季/淡季区别）
- 门票价格（成人票、学生票、老人票等）
- 建议游玩时长
- 最佳旅行季节
- 景区官网链接

### 3.3 交通指南
- 外部交通: 飞机、高铁/火车、长途汽车如何抵达
- 内部交通: 景区内观光车、索道、游船等
- 从市区到景区的具体路线（含打车预估费用）
- 地图嵌入 / 路线示意图

### 3.4 推荐游览路线
- 一日游精华路线（含时间节点建议）
- 两日游深度路线
- 亲子/老人友好路线（标注）
- 每个路线附总步行距离和体力消耗评级

### 3.5 美食推荐
- 景区周边必尝美食（含参考价格）
- 推荐餐厅列表（名称、特色菜、人均消费）
- 当地特色小吃介绍

### 3.6 住宿推荐
- 按预算分档: 经济型 / 舒适型 / 豪华型
- 推荐酒店名称及大致价格区间
- 建议住宿区域（靠近景区 / 市区交通便利）

### 3.7 实用贴士
- 注意事项（如高反、天气、着装建议）
- 最佳拍照机位推荐
- 避坑指南（常见消费陷阱）
- 周边联游景点推荐

### 3.8 用户互动
- 评论区 / 留言板
- 评分功能
- 收藏 / 喜欢

## 4. 网站功能模块

### 4.1 首页
- 十大景点卡片式展示（缩略图 + 名称 + 评分 + 一句话简介）
- 按地区/热度/主题筛选（如 "自然风光"、"历史文化"、"城市休闲"）
- 搜索框（支持景点名、城市关键词搜索）
- 热门排行榜边栏

### 4.2 景点详情页
- 左右分栏布局（左侧主内容 / 右侧快速导航小卡片）
- 信息模块可折叠/展开，默认展开核心模块
- 侧边悬浮目录（TOC）锚点导航
- "收藏" 和 "分享" 按钮
- 底部相关景点推荐（同地区/同类主题）

### 4.3 对比功能
- 勾选 2-3 个景点加入对比
- 对比表格: 门票、最佳季节、游玩时长、交通便利度等维度

### 4.4 全局功能
- 响应式布局（桌面 / 平板 / 手机）
- 中英文切换
- 深色/浅色主题切换
- 面包屑导航
- 页脚: 关于我们、免责声明、联系方式

## 5. 技术栈建议

| 类别 | 推荐方案 | 说明 |
|------|---------|------|
| 前端框架 | React + Vite 或纯 HTML/CSS/JS | 根据复杂度选择 |
| 样式方案 | Tailwind CSS | 快速构建，响应式友好 |
| 路由 | React Router (如用 React) | 多页面管理 |
| 图标 | Lucide Icons 或 Heroicons | 丰富图标库 |
| 地图 | 高德地图 JS API / Leaflet | 景点位置标注 |
| 图片 | Unsplash / 本地静态资源 | 景点配图 |
| 部署 | Vercel / GitHub Pages | 静态站点友好 |

## 7. 非功能性需求

- **加载性能:** 首屏加载 < 2s，图片懒加载
- **SEO:** 每个页面独立 title / meta description / 结构化数据
- **可访问性:** 语义化 HTML，Alt 文本，键盘导航支持
- **兼容性:** 支持主流浏览器最新两个版本
- **内容维护:** 攻略数据与页面逻辑分离，便于后续更新内容

## 8. 交付物

- 完整 HTML/CSS/JS 源代码（或 React 项目源码）
- 景点攻略内容数据文件（JSON 格式，方便维护）
- 所有页面在桌面和移动端的正常展示
- 部署说明文档
